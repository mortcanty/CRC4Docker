Canberra Distance with Location Parameter
-----------------------------------------
See [Jurman08]_.

.. autofunction:: canberra_location(x, y, k=None)

The function computes:
   .. math::
   
      \sum_i{\frac{|\min\{x_i+1, k+1\} - \min\{y_i+1, k+1\}|}
         {\min\{x_i+1, k+1\} + \min\{y_i+1, k+1\}}}
